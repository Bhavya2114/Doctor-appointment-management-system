3) Why do you write mongoose.models.user || mongoose.model(...)?

Mongoose throws an OverwriteModelError if we try to define the same model multiple times. So we first check if the model already exists inside mongoose.models. If it exists, we reuse it. Otherwise, we create it. This prevents duplicate model compilation errors during development or hot reload.
